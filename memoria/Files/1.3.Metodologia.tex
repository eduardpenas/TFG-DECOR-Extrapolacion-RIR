\section{Implementación Técnica}

Para asegurar la validez de los resultados, se han seguido los siguientes criterios técnicos:

\begin{itemize}
	\item \textbf{Simulación:} Se utiliza el método de fuentes virtuales (\textit{Image Source Method}) a través de la librería Pyroomacoustics para obtener datos de entrenamiento fiables.
	\item \textbf{Arquitectura de Red:} Se emplea una red LSTM (Long Short-Term Memory) debido a su eficacia para modelar el decaimiento temporal del sonido. Además, se integran capas de atención para capturar dependencias de largo alcance en la señal.
	\item \textbf{Función de Pérdida:} Se utiliza una pérdida basada en el espectrograma (MSTFT) para optimizar la señal tanto en tiempo como en frecuencia.
	\item \textbf{Reconstrucción:} Se aplica el método \textit{Random Sign-Sticky} para garantizar que la fase de la señal reconstruida sea físicamente plausible \cite{muhammad2025}.
\end{itemize}

\section{Herramientas de Software}

El desarrollo se realiza íntegramente en Python, utilizando PyTorch como motor de aprendizaje profundo y NumPy para el análisis estadístico de los datos acústicos obtenidos.
